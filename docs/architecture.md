# PokemonSimulator еӨ§ж•°жҚ®еұ•зӨәе№іеҸ° - жһ¶жһ„ж–ҮжЎЈ

## жҰӮиҝ°

PokemonSimulator жҳҜдёҖдёӘ C++17 Pokemon Gen9 еҚ•жү“еҜ№жҲҳжЁЎжӢҹеҷЁпјҢй…ҚеӨҮе®Ңж•ҙзҡ„ Web еүҚеҗҺз«ҜгҖӮжң¬е№іеҸ°еңЁжӯӨеҹәзЎҖдёҠжһ„е»әеӨ§ж•°жҚ®еұ•зӨәеұӮпјҢе®һзҺ°пјҡ
1. **йқҷжҖҒж•°жҚ®**пјҲPokemon Showdown зҲ¬еҸ–зҡ„ JSONпјүвҶ’ Hadoop HDFS вҶ’ PySpark жү№еӨ„зҗҶ CRUD + еҲҶжһҗ
2. **жөҒејҸж•°жҚ®**пјҲеүҚеҗҺз«ҜеҜ№жҲҳдәӨдә’пјүвҶ’ Kafka ж¶ҲжҒҜйҳҹеҲ— вҶ’ PySpark Streaming е®һж—¶иҒҡеҗҲ вҶ’ Redis зј“еӯҳ
3. **зј“еӯҳдёҺеҚҸи°ғ**пјҡRedis йӣҶзҫӨпјҲзј“еӯҳ/жҺ’иЎҢжҰң/йҷҗжөҒпјүгҖҒZooKeeperпјҲKafka еҚҸи°ғ / Spark HA / еҲҶеёғејҸй”Ғпјү
